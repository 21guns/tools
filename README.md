# tools 开发中相关工具

## 项目介绍
项目开发中提高效率的工具，python shell等工具

### exceltojava
根据excel生产java代码
`python3 excelToClass.python`

### filestofile.python
多个文件合成一个文件
`python3 filestofile.python`

### gl.sh
更新当前目录下的所有git项目
`sh gl.sh`

### release.sh
将git的uat分支合并到master分支，并在master打tags
`sh release.sh v2.3.15`

### sqlAddAs.sh
`sh sqlAddAs.sh 'table_alise' 'column_allise' 'columns'`
`sh sqlAddAs.sh 'q' 'q_' ''`
